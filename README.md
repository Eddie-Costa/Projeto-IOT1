# Projeto-IOT1
Simulação de sensor de movimento usando Python e interface web. O script Sensor.py detecta movimento do mouse com pynput e atualiza o estado do sensor. A página sensor.html lê esse estado e faz o círculo piscar. Execução: 1️⃣ Instale pynput 2️⃣ Rode Sensor.py 3️⃣ Abra sensor.html no navegador.
